# Personal Portfolio — ISHIMWE Lina Assoumani

A modern, dark-mode personal portfolio built with **Next.js 15**, **TypeScript**, and **Tailwind CSS**.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deploy on Vercel

1. Push this repo to GitHub
2. Import the project at [vercel.com/new](https://vercel.com/new)
3. Vercel will auto-detect Next.js — click Deploy

## Customize

- **Photo:** Replace the placeholder in `src/components/About.tsx` with your image (add to `public/` and use `next/image`)
- **SafeZone GitHub:** Add the link in `src/data/portfolio.ts` when ready
- **Content:** All copy lives in `src/data/portfolio.ts`

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- Lucide React (icons)
